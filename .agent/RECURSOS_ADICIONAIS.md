## 📦 Como Instalar Novos Recursos

### Método 1: Via Marketplace (Recomendado)

1. Abra o Antigravity
2. Acesse o Marketplace de agentes/skills
3. Busque pelo nome do recurso
4. Clique em "Install"

### Método 2: Manual

1. Baixe o arquivo `.md` do recurso
2. Coloque na pasta apropriada:
   - Agentes: `.agent/agents/`
   - Skills: `.agent/skills/`
   - Workflows: `.agent/workflows/`

## 💡 Dicas de Instalação

### ✅ Antes de Instalar

1. **Leia a documentação** do recurso
2. **Verifique compatibilidade** com seu projeto
3. **Revise exemplos** de uso
4. **Confira reviews** da comunidade

### ⚠️ Cuidados

1. **Não instale tudo de uma vez** - Teste gradualmente
2. **Mantenha backup** antes de grandes mudanças
3. **Teste em ambiente de desenvolvimento** primeiro
4. **Monitore performance** após instalação

### 🎯 Otimização

1. **Desinstale recursos não usados** - Mantenha apenas o necessário
2. **Atualize regularmente** - Novos recursos e correções
3. **Combine recursos** - Use agentes + skills juntos
4. **Documente uso** - Anote como cada recurso ajuda

# 🤖 Agente OpenRouter

Este agente foi configurado para atuar como uma ponte para os modelos de fronteira disponíveis no OpenRouter (como Claude 3.5 Sonnet, GPT-4o, DeepSeek, etc.).

## 🎯 Quando usar

- Quando precisar de raciocínio logicamente complexo.
- Quando quiser comparar respostas entre diferentes modelos.
- Para tarefas que exigem uma janela de contexto maior ou capacidades específicas de modelos de terceiros.

## 🛠️ Ferramentas Disponíveis

O agente utiliza a ferramenta do servidor MCP `openrouter` configurada em seu sistema.

## 💡 Como chamar

"Use o agente OpenRouter para [tarefa]"

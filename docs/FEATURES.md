## ⚡ Sistema de Orquestração (Eco Mode)

### Integração com IA

- Acesso a modelos de IA gratuitos
- Orquestração inteligente de tarefas
- Fallback automático entre modelos
- Persistência de configurações

### Modelos Disponíveis

- **Tier S**: DeepSeek R1, Llama 3.3 70B
- **Tier A**: Gemini Flash Lite, Mistral Nemo
- **Tier B**: Modelos leves e rápidos

### Comandos Disponíveis

- `/enable-eco`: Ativa modo econômico
- `/disable-eco`: Desativa modo econômico
- `/set-model`: Define modelo específico
- `/orchestrate`: Executa tarefas complexas
